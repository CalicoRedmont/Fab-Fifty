# FAB FIFTY - BAD PONG APERO SESSION

Mini-jeu HTML5 retro pour les 50 ans de Fabien.

## Jouer en ligne

Une fois GitHub Pages active, le jeu sera disponible ici :

https://calicoredmont.github.io/Fab-Fifty/

## Acces

Le jeu affiche un ecran d'acces. Le mot de passe accepte correspond au prenom d'un joueur du jeu, sans tenir compte des majuscules ni des accents.

Exemples :
- `Fabien`
- `Stephane` ou `Stéphane`
- `Cecile` ou `Cécile`
- `Olivier`

## Local

Le jeu peut aussi etre lance localement en ouvrant `index.html` dans un navigateur moderne.
